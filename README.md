<h2>📖 About</h2>
<p>This project is a simple <strong>Product Management CRUD application</strong> developed using <strong>ASP.NET Core MVC</strong> and <strong>Entity Framework Core</strong>. It demonstrates how to implement basic database operations in a .NET web application.</p>

<h2>⚙️ Installation</h2>

<p>Clone the repository:</p>

<pre><code>git clone https://github.com/yourusername/yourrepo.git</code></pre>

<p>Navigate to the project folder and run:</p>

<pre><code>dotnet restore</code></pre>

<h2>🚀 Usage</h2>

<p>Run the project with the following command:</p>

<pre><code>dotnet run</code></pre>

<p>After running, open your browser and go to:</p>

<pre><code>https://localhost:5001</code></pre>

<h2>🧩 Features</h2>

<ul>
<li>Product listing</li>
<li>Create new products</li>
<li>Edit existing products</li>
<li>ASP.NET Core MVC architecture</li>
<li>Entity Framework Core database integration</li>
</ul>
